# ЯСНА — Чек-лист верификации и сводная таблица обоснований

## ЧАСТЬ I. УНИВЕРСАЛЬНЫЙ ЧЕК-ЛИСТ РАЗМЕЩЕНИЯ

Для проверки правильности размещения элемента X на полочку N необходимо пройти **все 7 уровней проверки**. Элемент размещён верно **только при 7 из 7**. Если хотя бы один уровень не пройден — размещение ошибочно, элемент нужно переместить или переформулировать.

---

### УРОВЕНЬ 1: ПРОВЕРКА ТИПА (долгое / короткое)

**Правило:** Чётные позиции (0,2,4,6,8,10) = ДОЛГОЕ. Нечётные (1,3,5,7,9,11) = КОРОТКОЕ.

| Вопрос | Если ДА → | Если НЕТ → |
|--------|-----------|------------|
| X — это протяжённый процесс, плавное накопление, постепенное изменение? | Чётная позиция | Нечётная |
| X — это резкий перелом, быстрая перемена, точка перехода? | Нечётная позиция | Чётная |
| X — это пространство/зона (а не точка/момент)? | Чётная | Нечётная |
| X — это граница, порог, вход/выход? | Нечётная | Чётная |

**Примеры из Ясны Суток:**
- Утренняя заря (2) — долгий процесс нарастания света ✓
- Выход Солнца (3) — мгновенный перелом, 2 минуты ✓
- Дневной переход (6) — длительный процесс ✓
- Первая тьма (7) — момент перелома ✓

---

### УРОВЕНЬ 2: ПРОВЕРКА ПРАНЫ (стихии)

**Правило:** Прана определяется формулой `index % 4`:
- 0,4,8 = ШЭ (Земля) — **устойчивость, неизменность, фундамент**
- 1,5,9 = ФО (Вода) — **спад, понижение, текучесть, холод**
- 2,6,10 = ЦИ (Воздух) — **неустойчивый покой, лёгкость, переход**
- 3,7,11 = ХА (Огонь) — **рост, повышение, вспышка, жар**

| Вопрос | ШЭ (0,4,8) | ФО (1,5,9) | ЦИ (2,6,10) | ХА (3,7,11) |
|--------|------------|------------|-------------|-------------|
| X устойчиво, неподвижно, определённо? | ✓ | | | |
| X стекает, спадает, остывает, понижается? | | ✓ | | |
| X легко, воздушно, переходно, промежуточно? | | | ✓ | |
| X горячо, растёт, вспыхивает, острое? | | | | ✓ |
| X — основа, фундамент, почва? | ✓ | | | |
| X — исток, родник, источник, начало потока? | | ✓ | | |
| X — канал, русло, проводник, несущая среда? | | | ✓ | |
| X — искра, пламя, взрыв, рождение? | | | | ✓ |

**Связка с типом:** ШЭ и ЦИ → всегда долгие. ФО и ХА → всегда короткие.

---

### УРОВЕНЬ 3: ПРОВЕРКА КРЕСТА (роль в цикле)

**Правило:**
- Опорный (0,3,6,9) — **Основа/Бой**: главные события, ось бытия
- Правый (1,4,7,10) — **Исход/Победа**: результат предыдущего, итог
- Левый (2,5,8,11) — **Вера/Подготовка**: подготовка к следующему

| Вопрос | Опорный (0,3,6,9) | Правый (1,4,7,10) | Левый (2,5,8,11) |
|--------|--------------------|--------------------|-------------------|
| X — это само событие, основа, база? | ✓ | | |
| X — это результат чего-то, итог, исход? | | ✓ | |
| X — это подготовка к чему-то, ожидание? | | | ✓ |
| X — это то, без чего ничего не будет? | ✓ | | |
| X — это то, что получается ПОСЛЕ действия? | | ✓ | |
| X — это то, что делается ПЕРЕД действием? | | | ✓ |

**Ритм проверки:** Вера(L) → Бой(O) → Победа(R), повторяется 4 раза:
```
2(Вера) → 3(Бой) → 4(Победа)
5(Вера) → 6(Труд) → 7(Победа)
8(Вера) → 9(Бой) → 10(Победа)
11(Вера) → 0(Ожидание) → 1(Победа)
```

---

### УРОВЕНЬ 4: ПРОВЕРКА ПРОТИВОПОЛОЖНОСТИ

**Правило:** Позиция N и позиция (N+6)%12 должны быть **полными противоположностями** по смыслу.

| Пара | Принцип противоположности |
|------|---------------------------|
| 0 ↔ 6 | Тьма ↔ Свет. Вход ↔ Центр. Начало ↔ Вершина. Грязное ↔ Чистое. |
| 1 ↔ 7 | Первое ↔ Последнее (в своём роде). Холод ↔ Жар. Начало света ↔ Начало тьмы. |
| 2 ↔ 8 | Нарастание ↔ Спадание (один процесс в разных направлениях). |
| 3 ↔ 9 | Истина ↔ Розыгрыш. Проявление ↔ Сокрытие. Восток ↔ Запад. |
| 4 ↔ 10 | Утренний итог ↔ Вечерний итог. Рост ↔ Угасание. |
| 5 ↔ 11 | Конец тьмы ↔ Конец света. Одно закончилось ↔ другое закончилось. |

**Чек:** Если X стоит на позиции N, то элемент на (N+6)%12 должен быть зеркально противоположен X. Если не удаётся сформулировать противоположность, размещение неверно.

---

### УРОВЕНЬ 5: ПРОВЕРКА СОСЕДСТВА (последовательность)

**Правило:** Элемент X на позиции N должен логически **следовать из** элемента на позиции N−1 и **вести к** элементу на позиции N+1.

**Механизм:** В долгой позиции (чётной) что-то НАКАПЛИВАЕТСЯ. В следующей короткой (нечётной) накопленное ПЕРЕХОДИТ в новое качество.

| Переход | Вопрос для проверки |
|---------|---------------------|
| (N−1) → N | X логически следует из предыдущего? |
| N → (N+1) | Из X логически вытекает следующее? |
| Долгое → Короткое | Накопленное в долгом переходит в качественный скачок в коротком? |
| Короткое → Долгое | После скачка начинается новое накопление? |

---

### УРОВЕНЬ 6: ПРОВЕРКА ПОЛОВИНЫ (верхняя/нижняя, левая/правая)

**Правило:**

| Половина | Позиции | Качество |
|----------|---------|----------|
| Верхняя (День) | 4,5,6,7,8 | Явное, открытое, видимое, активное, общественное |
| Нижняя (Ночь) | 10,11,0,1,2 | Скрытое, закрытое, невидимое, пассивное, личное |
| Левая (Нарастание) | 1,2,3,4,5 | Рост, начало, восхождение, утро |
| Правая (Спадание) | 7,8,9,10,11 | Спад, завершение, нисхождение, вечер |

**Чек:** X на верхней половине — должен быть «явным». X на нижней — «скрытым». X слева — «нарастающим». X справа — «спадающим».

---

### УРОВЕНЬ 7: ПРОВЕРКА ЯЗЫКОВОЙ СВЯЗИ

**Правило:** Элементы на одной полочке в РАЗНЫХ Яснах должны быть связаны через русский язык, метафоры или функциональные аналогии.

**Чек:** Можно ли найти пословицу, поговорку или метафору, связывающую X с элементами на той же полочке в Ясне Суток, Ясне Двора, Ясне Дома?

## ЧАСТЬ II. СВОДНАЯ ТАБЛИЦА ОБОСНОВАНИЙ

Ниже для каждой полочки (0–11) приведены элементы из всех изученных Ясен с обоснованием, ПОЧЕМУ они именно на этой полочке.

### ПОЛОЧКА 0 — ШЭ (Земля), Опорный крест, Долгое

**Функция:** ВХОД / ОСНОВА / ТЬМА / ГРЯЗЬ / НАЧАЛЬНАЯ ТОЧКА

| Ясна | Элемент | Обоснование |
|------|---------|-------------|
| Суток | Тёмная ночь | Сила Тьмы. Устойчивое состояние (ШЭ). Основа (Опорный). Нет изменений (долгое). |
| Двора | Ворота | Вход во двор. Основа доступа. Здесь грязь от телег (ШЭ=земля). |
| Животных | Свинья | Любит грязь (ШЭ=земля). Максимально далеко от дома. Козерог=Козелог=земляная улитка. |
| Дома | Прихожая | Вход в дом. «Свинство» с улицы (грязная обувь). Коридор = ночной проход. |
| Кухни | Вход/Фартук | Вход на кухню. Защита от свинства. Начальная точка процесса. |
| Круговорота | Грязь (вода+земля) | Буквально ШЭ=Земля + ФО=Вода = грязь. Самая нижняя точка цикла. |
| Спальни | Халат | Вход в спальню, первое что надеваем/снимаем. Защита (как фартук). |
| Тела | Ступни | Самый низ тела. Контакт с землёй (ШЭ). Основа опоры. |
| Завода | Рекламный отдел | «Ворота» предприятия для внешнего мира. Вход клиента. |
| Облачности | Свинцовое небо | Полная облачность. «Тёмная ночь» неба. Устойчивое (ШЭ) состояние. |
| Месяцев | Январь (Стужень) | Самый тёмный, холодный месяц. Полная тьма года. |
| Головы | Шея | Основание головы. Опора (ШЭ). Вход из тела в голову. |
| Театра | Коридор | Вход, начальный проход. Тёмный коридор. |
| Колокольни | Река/Колодец | Основание. Самый низ. Вода = исток. |

**Языковые связи:** Свинья → свинство → грязь → ворота → прихожая → грязные сапоги. Козерог → козелог → земляная улитка → земля → грязь.

### ПОЛОЧКА 1 — ФО (Вода), Правый крест, Короткое

**Функция:** ПЕРВЫЙ РЕЗУЛЬТАТ / ИСКРА / ИСТОК / ГОСТЬ / САМЫЙ ХОЛОД

| Ясна | Элемент | Обоснование |
|------|---------|-------------|
| Суток | Первый свет | Первый результат (Правый) ожидания во тьме. Искра. Самый холод (ФО=вода=холод). |
| Двора | Калитка | Результат прохода (после ворот). Маленький вход. «Водичка дырочку найдёт» (ФО). |
| Животных | Грызуны | Прогрызут путь, как вода (ФО). Первые незваные гости. Кролики. |
| Дома | Гостиная | Результат прихода (после прихожей). Место для гостей (калитка → гости). «Перемывать кости» (грызуны). |
| Кухни | Мойка | Вода (ФО буквально). Первый этап обработки продуктов. «Водолей». |
| Круговорота | Болото/Родник/Ключ | Исток воды (ФО). Первый свет → первый ключ. Болото = много луж (результат накопления). |
| Спальни | Гантели/Вода | Графин с водой (ФО). Первое после пробуждения. |
| Тела | Половые органы | Исток жизни. Вода (ФО). Самый «холодный» по расположению. |
| Завода | Отдел кадров/закупок | Первый приток ресурсов. Снабжение = поток (ФО). |
| Облачности | Первые неровности | Первое изменение после сплошного неба. Появление текстуры. |
| Месяцев | Февраль (Лютый) | Самый лютый холод (ФО=холод). Первые признаки весны — искра. |

**Языковые связи:** Водолей → вода → мойка → ключ → родник. Грызуны → «грызть», «перемывать косточки» → гостиная (болтовня).

### ПОЛОЧКА 2 — ЦИ (Воздух), Левый крест, Долгое

**Функция:** КАНАЛ / ПОТОК / НАРАСТАНИЕ / ВЕРА / ПОДГОТОВКА К БОЮ

| Ясна | Элемент | Обоснование |
|------|---------|-------------|
| Суток | Утренняя заря | Неявный свет нарастает (ЦИ=воздух=лёгкость). Лёгкое время. Вера в победу (Левый). |
| Двора | Конюшня | Лошадь несёт, перевозит (ЦИ=канал). Подготовка к действию. |
| Животных | Лошадь | «Несёт» = канал доставки (ЦИ). «Жрать как лошадь» → столовая. |
| Дома | Столовая | Еда приносится из кухни (канал). «Кормить сказками» = две рыбы (Рыбы ♓). |
| Кухни | Разделочный стол | Подготовка продуктов (Левый=подготовка). Процесс обработки (долгое). |
| Круговорота | Река | Несёт (как лошадь). Канал (ЦИ). Долгий процесс течения. |
| Спальни | Зарядка | Подготовка тела (Левый=подготовка). Воздух (ЦИ) = дыхание при зарядке. |
| Тела | Живот | Переработка пищи. Канал пищеварения. Нарастание (наполнение). |
| Завода | Рабочие | Производственный поток. Канал создания ценности. Подготовка продукции. |
| Месяцев | Март (Березень) | Нарастание тепла. Вера в весну. Березы «начинают течь» (сок). |

**Языковые связи:** Рыбы ♓ → две рыбы → «кормить сказками» → столовая. Лошадь → «несёт» → река → канал.

### ПОЛОЧКА 3 — ХА (Огонь), Опорный крест, Короткое

**Функция:** ГЛАВНОЕ СОБЫТИЕ / ОГОНЬ / РОЖДЕНИЕ / ГОРИЗОНТ / ИСТИНА

| Ясна | Элемент | Обоснование |
|------|---------|-------------|
| Суток | Выход Солнца | Проявление Света! Огонь (ХА). Опорное событие. Стрелка Весов = Истина. 2 минуты = короткое. |
| Двора | Козлы/Бараны | Бодание = борьба на горизонте (Опорный=бой). Барашки (волны). |
| Животных | Козлы/Бараны | Бодаются = сражение (Опорный). Овен ♈ = огонь. |
| Дома | Кухня | Огонь! (ХА). Кухня=Кузня (одно слово через букву Кси). Рождение блюда. |
| Кухни | Плита | Буквально Огонь (ХА). Главное событие кухни. Опорное. |
| Круговорота | Поверхность водоёма | Горизонт воды! Граница вода/воздух (Опорный=линия борьбы). «Барашки» = козлы. |
| Спальни | Одеяло/Сундук | Греет (ХА=огонь). Или обогреватель. |
| Тела | Грудь | Огонь сердца (ХА). Грудь = горизонт тела. |
| Завода | Плановый отдел | «Бой за план» (Опорный=бой). Рождение плана. |
| Месяцев | Апрель (Квитень) | Цветение = вспышка (ХА). Весеннее пробуждение. |

**Языковые связи:** Кухня ↔ кузня (через Кси). Козлы → бодаются → борьба. Барашки (облака) = барашки (волны). Овен = огонь.

### ПОЛОЧКА 6 — ЦИ (Воздух), Опорный крест, Долгое

**Функция:** ВЕРШИНА / ЦЕНТР / СВЕТ / ГЛАВНЫЙ ПРОЦЕСС / СОЮЗ

| Ясна | Элемент | Обоснование |

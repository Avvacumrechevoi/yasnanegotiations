## Эталон

Тема **T1 «Что есть Ясна»** реализована полностью как пример формата для остальных. Внутри:

— **5 категорий** (Сущность Ясны / Веда-Гимн-Повесть-Памятник / Язык Веды / Следы Ясны в русском / Корпус и метод);
— **32 атома** 9 видов (definition · fact · rule · postulate · name · pairing · etymology · list · number);
— **10 вопросов** разных типов (single-choice, multi-choice, fill-blank, true-false, match-pair);
— **8 отброшенных пунктов** с явным обоснованием (повторы, рекапы, мета-комментарии, переходный анонс);
— **coverage 80%** = 33 покрытых пункта / (33 + 8 отброшенных);
— все цитаты с `source.items` (номера пунктов) и `source.pdf_page`;
— перекрёстные ссылки в `meta.links` (5 типов: see_also, prerequisite, leads_to, contrasts, applies_in).

## Принципы

1. **Книга — мастер.** Все цитаты дословно. Пункты `[№N]` неприкосновенны.
2. **ID стабильны.** `T1.K1.A1` навсегда указывает на «Ясна есть способ описания природы».
3. **Каждый атом — самостоятельный факт.** Не пересказ, не интерпретация.
4. **Дистракторы правдоподобны.** Берутся из самой книги или из общеизвестного, никаких выдумок.
5. **Метаданные позволяют авто-генерировать вопросы:** `kind` атома → подходящие `type` вопросов.

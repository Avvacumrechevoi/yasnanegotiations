# ✦ Ясна-Конструктор

Инструмент для создания и изучения 12-лучевых звёздных диаграмм по системе Ясны.

## 🌐 [Открыть приложение](https://avvacumrechevoi.github.io/yasnanegotiations/)

## Архитектура

```
├── docs/index.html    — Приложение (React SPA, GitHub Pages)
├── backend/           — Express API (отдельный деплой)
├── specifications/    — Спецификации (6 документов)
└── .github/workflows/ — Автодеплой на Pages
```

## Возможности

- **29 шаблонов Ясен** из рубрикатора, уроков и PDF
- **11 механик** с визуальным наложением (мульти-выбор)
- **Совмещение** двух Ясен на одной звезде
- **Контекстная карточка** с описанием позиции и механик
- **Глоссарий** — 12 карточек с объяснением каждой механики
- **Инструкция** — 7 шагов составления + 3 чек-листа
- **Редактор** — создание новых Ясен

## Стиль

Apple Light Theme: белый фон, системные шрифты, синий акцент (#0071e3).
